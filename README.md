# Customer Segmentation using Clustering
This project applies clustering techniques to perform customer segmentation based on numerical features.
# Overview
The goal is to group customers into distinct segments to better understand patterns in the data.
# Methods
- Data processing:
  - Removed non-numerical features
  - Handled missing values
  - Standardized features using StandardScaler
- Clustering:
  - K-Means (with eblow method to determine optimal number of clusters)
  - DBSCAN (parameter tuning for density-based clustering)
- Evaluation:
  - Silhouette score
# Tech Stack
- Python
- Pandas, Numpy
- Scilit-learn
# Notes
This project was completed as part of coursework and demonstrates basic skills in unsupervised learning and data preprocessing.
